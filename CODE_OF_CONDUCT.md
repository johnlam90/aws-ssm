# Code of Conduct

We follow the [Contributor Covenant](https://www.contributor-covenant.org/) v2.1.

## Our Pledge
We pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards
Examples of positive behavior include welcoming language, respectful feedback, focus on what is best for the community.

Unacceptable behavior includes harassment, trolling, discriminatory remarks, publishing private information, or other conduct not aligned with the Covenant.

## Enforcement
Report unacceptable behavior to the maintainers via a GitHub security advisory or email listed in SECURITY.md. All complaints will be reviewed promptly and fairly.

This Code of Conduct is adapted from the Contributor Covenant version 2.1.
